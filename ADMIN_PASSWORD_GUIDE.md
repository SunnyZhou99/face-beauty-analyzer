## 修改密码

### 方法 1: 修改环境变量（推荐）

1. 编辑 `.env.local` 文件
2. 找到这一行：
   ```
   NEXT_PUBLIC_ADMIN_PASSWORD=admin123
   ```
3. 修改为你想要的密码，例如：
   ```
   NEXT_PUBLIC_ADMIN_PASSWORD=mySecurePassword123
   ```
4. 保存文件
5. 重启开发服务器：`npm run dev`

### 方法 2: 在 Netlify 上修改

1. 登录 Netlify
2. 选择你的项目
3. 点击 `Site settings` → `Environment variables`
4. 添加或更新环境变量：
   - **Key**: `NEXT_PUBLIC_ADMIN_PASSWORD`
   - **Value**: 你的新密码
5. 保存后触发重新部署

## ⚠️ 安全提示

### 当前方案的局限性

1. **密码存储在环境变量中**: 虽然比硬编码安全，但仍需谨慎
2. **没有加密**: 密码以明文形式存储
3. **没有会话超时**: 登录后一直有效，直到点击退出登录或清除浏览器缓存

### 提高安全性的建议

#### 1. 使用更强的密码

修改为更复杂的密码：
```
NEXT_PUBLIC_ADMIN_PASSWORD=My@Str0ng!P@ssw0rd#2024
```

#### 2. 定期更换密码

建议每个月更换一次管理后台密码。

#### 3. 限制访问 IP（可选）

可以在 Netlify 设置中限制管理后台页面的访问 IP。

#### 4. 使用 Supabase Auth（高级）

对于更高的安全性，可以使用 Supabase Auth 来实现真正的用户认证系统。
